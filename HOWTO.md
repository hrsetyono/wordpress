# Site Name

Tested working on:

- WordPress 5.4
- PHP 7.3
- MySQL 8.0
- Server nginx

## Setup Guide

Requirement:

- [WAMP Server with](https://github.com/hrsetyono/wordpress/wiki/Installing-WAMP-on-Windows-10) with PHP 7.0+ and MySQL database
- Install [Composer](https://getcomposer.org/download/)
- Install [WP CLI](https://github.com/hrsetyono/wordpress/wiki/Installing-WP-CLI-on-Windows-10)

Setup local development:

1. Run `composer update` to download plugins.
1. Run `composer wp-init` to download WordPress core files and setup database.
